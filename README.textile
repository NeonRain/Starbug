h1. Starbug PHP Web Development Framework

Starbug PHP is an open source framework for PHP. Starbug is a fast and exstensible toolkit that employs a Model-View design pattern. Here are some of the core features SB provides:

* Templating
* Role-based access control (RBAC)
* Database management and abstraction
* Data validation
* XSLT 2.0 code generation
* Utilities for data grids, form building, pagination, tagging, thumbnailing and more..
* Plugins and hooks built on a flexible publish/subscribe layer

h2. Dependencies

You will need the following to use this toolkit:

* "PDO extension for PHP":http://php.net/manual/en/book.pdo.php
* "Saxon XSLT Processor":http://saxon.sourceforge.net (requires Java Runtime Environment)

h2. Getting Started

Install the "Starbug PHP Installer":http://github.com/cogentParadigm/Starbug/tree/installer and then follow these steps to get Starbug set up:

# Initialize a new application and change to that directory:<pre><code>starbug myapp
cd myapp</code></pre>
# Enter your database and other details into <code>etc/Host.php</code>.
# Run setup:<pre><code>sb setup</code></pre>
# Open up your browser and point it to your installation directory!
