h1. Starbug PHP Web Development Framework

Starbug PHP is an open source framework for PHP. Starbug is a fast and exstensible toolkit that employs a Model-View paradigm. Here are some of the core features SB provides:

* Templating
* Role-based access control (RBAC)
* Database management and abstraction
* Data validation
* XSLT 2.0 code generation
* Utilities for data grids, form building, pagination, tagging, thumbnailing and more..
* Plugins and hooks built on a flexible publish/subscribe layer

h2. Getting Started

Follow these steps to get Starbug set up:

# Clone a copy from github, and change to that directory:<pre><code>
git clone git://github.com/cogentParadigm/Starbug.git starbug
cd starbug</code></pre>
# If this is the first time you are installing Starbug, copy <pre>core/script/sb</pre> into your path, and <pre>core/script/sb_completion</pre> into your bash_completion directory:<pre><code>
sudo cp core/script/sb /usr/bin/
sudo chmod +x /usr/bin/sb
sudo cp core/script/sb_completion /etc/bash_completion.d/sb</code></pre>
# Enter your database and other details into <code>etc/Host.php</code>.
# Run setup:<pre><code>
sb setup</code></pre>
# Open up your browser and point it to your installation directory!
